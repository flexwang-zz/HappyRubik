Happy Rubic 欢乐魔方
=========
Happy Rubic is an app of Rubik's Cube on Android, which implements almost everything about Rubik's Cube.


欢乐魔方是一款安卓上的魔方应用。它几乎完美支持三阶魔方的各种操作。
  - 单指旋转一列魔方进行单列旋转；

  - 双指旋转两列魔方进行双列旋转；

  - 双指旋转一列魔方进行180°旋转；

  - 在经典模式下摇一摇可以打乱或还原；

  - 在计时模式和对战模式下可以保存录像；
